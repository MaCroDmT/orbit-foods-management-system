# 🍞 Orbit Foods – Bakery Management & E-commerce System

A full-stack **Bakery Management System** built with **Django**, designed to streamline bakery operations and provide a seamless online shopping experience for customers.

---

## 🚀 Features

### 🛍️ Customer Side

* Browse products (Store)
* Add to cart 🛒
* Place orders
* User authentication (Login/Register)
* Order history tracking

### 🧑‍💼 Admin Panel

* Admin dashboard
* Product management
* Order management
* Admin/user control system

### 🎨 UI/UX

* Responsive design using **Bootstrap**
* Clean modern interface
* Dynamic navbar with user roles
* Flash messaging system

---

## 🧱 Tech Stack

* **Backend:** Django
* **Frontend:** HTML, CSS, Bootstrap
* **Database:** MySQL
* **Authentication:** Django Auth System

## ⚙️ Installation Guide

```bash
git clone https://github.com/your-username/orbit-foods.git
cd orbit-foods

python -m venv venv
venv\Scripts\activate   # or source venv/bin/activate

pip install -r requirements.txt

# Setup MySQL database manually

python manage.py migrate
python manage.py runserver
```

---

## 🔐 Environment Setup

Create a `.env` file:

```env
DB_NAME=your_db_name
DB_USER=your_db_user
DB_PASSWORD=your_password
DB_HOST=localhost
```
